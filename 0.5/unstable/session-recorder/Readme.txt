Description:
	This module is used for recording the in-world activities along with the user's audio. 
	Before Using this module, kindly check the Pre-requisite and Optional Features of this module.
	
Pre-requisite:
	1) Install ffmpeg on the server side and give the absolute path in the FFMPEG Path field in Configuration in Role Play Recording on 
	2) Go to Openwonderland Server Admin -> Role Play Recordings -> Configuration,provide FFMPEG installation path (eg: /home/ubuntu/bin/ffmpeg).

Optional Features:
	If you want integration of vimeo, you have to provide the 'Access Token' and Default Group name.
	Default group is a group in which you want your video to be uploaded.
	You can enter this information on Openwonderland Server Admin -> Role Play Recordings -> Configuration.
